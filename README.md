# 🕊️ FlyHunter

**Subject**: Programming (PROG)  
**Name**: Abdanur Ayazbek

## 📌 Project Description

FlyHunter is a simple arcade-style game built with Pygame, where you control a bird to catch flying insects. The game features animated bird movement, dynamic fly behavior, and score tracking. Your goal is to "eat" as many flies as possible by colliding with them while flying across the sky.

The project is designed as a final task for the Programming course, showcasing skills in Python, Pygame, sprite animation, and collision detection.

## 💡 Features

- 🎮 Smooth mouse-based control for player movement  
- 🐦 Directional bird animation (left & right) using sprite sheets  
- 🦟 Random fly behavior with bounce-back edge collisions  
- 🌌 Background environment for immersive experience  
- 🧠 Menu system with mouse hover effects  
- 🧾 Score counter that updates on every successful fly catch  

## 📦 Technologies

- Python 3  
- Pygame library  
- Object-Oriented Programming  
- Basic Animation & Collision Systems  
- Sprite Management & Asset Handling  

## 🚀 How to Run

1. **Install dependencies**:  
   Make sure you have Python and Pygame installed:
   ```bash
   pip install pygame
   ```

2. **Clone the repository**:

   ```bash
   git clone https://github.com/MnstrsParago/PYGAME.FINAL.FlyHunter.git
   cd PYGAME.FINAL.FlyHunter
   ```

3. **Place required assets**:
   Ensure the following assets are available in the correct structure:

   ```
   ├── fly.png
   ├── backGoundAbda1.jpg
   └── left/
       ├── 1.png
       ├── 2.png
       ├── 3.png
       ├── 4.png
       └── 5.png
   ```

4. **Run the game**:

   ```bash
   python fly_hunter_game.py
   ```
