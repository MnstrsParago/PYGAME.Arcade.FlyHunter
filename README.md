# 🕊️ FlyHunter

**Subject:** Arcade Game Development (Pygame)
**Name:** Abdanur

## 📌 Project Description
**FlyHunter** — это простая аркадная игра на **Python + Pygame**, где игрок управляет птицей и ловит (съедает) мух, сталкиваясь с ними. В проекте реализованы базовые игровые механики: движение персонажа, анимация спрайтов, поведение врага/цели (мухи), подсчёт очков и простое меню.

## 💡 Features
- Управление птицей (плавное перемещение)
- Анимации движения (влево/вправо) на спрайтах
- Случайное движение мухи и отражение от границ экрана
- Подсчёт очков при успешном столкновении
- Фон и простая визуальная среда
- Меню (базовые UI-элементы)

## 📦 Technologies
- **Python 3**
- **Pygame**
- **OOP (Object-Oriented Programming)**
- Sprite Animation, Collision Detection

## 🚀 How to Run
1) Установи зависимости:
```bash
pip install pygame
```

2) Склонируй репозиторий:
```bash
git clone https://github.com/MnstrsParago/PYGAME.Arcade.FlyHunter.git
cd PYGAME.Arcade.FlyHunter
```

3) Запусти игру (пример):
```bash
python fly_hunter_game.py
```

> Если у тебя главный файл называется иначе (например `main.py`), просто замени команду на нужное имя файла.
